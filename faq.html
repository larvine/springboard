자주 묻는 질문 (FAQ)

궁금하신 내용을 확인해보세요

모델 설정 관련

어떤 AI 모델을 사용할 수 있나요?
다양한 AI 모델을 지원합니다. GPT-4, GPT-3.5, Claude, PaLM 등 주요 언어 모델을 사용하실 수 있으며, 각 모델의 특성에 맞게 선택하여 사용하실 수 있습니다.
모델 설정을 어떻게 변경하나요?
설정 페이지에서 '모델 설정' 메뉴를 선택하시면 됩니다. 여기서 사용하실 모델을 선택하고, 온도(temperature), 최대 토큰 수 등의 파라미터를 조정할 수 있습니다.
모델별 특징과 차이점은 무엇인가요?
각 모델은 고유한 특징을 가지고 있습니다. GPT-4는 더 정확하고 복잡한 작업에 적합하며, GPT-3.5는 빠른 응답 속도가 필요한 경우에 좋습니다. Claude는 긴 문맥을 처리하는데 강점이 있습니다.

키 설정 관련

API 키는 어떻게 발급받나요?
각 AI 서비스 제공업체의 공식 웹사이트에서 API 키를 발급받을 수 있습니다. OpenAI의 경우 platform.openai.com에서, Anthropic의 경우 console.anthropic.com에서 발급받으실 수 있습니다.
API 키를 어디에 입력하나요?
설정 페이지의 'API 키 관리' 섹션에서 입력하실 수 있습니다. 입력하신 키는 암호화되어 안전하게 저장됩니다. 필요한 경우 언제든지 수정하거나 삭제할 수 있습니다.
API 키가 유효하지 않다고 나옵니다
몇 가지를 확인해보세요: 1) 키를 정확하게 복사했는지 확인, 2) 키에 공백이나 특수문자가 포함되지 않았는지 확인, 3) 해당 서비스에서 키가 활성화되어 있는지 확인, 4) 계정에 충분한 크레딧이 있는지 확인하세요.

에러/타임아웃

타임아웃 에러가 자주 발생합니다
타임아웃은 여러 원인으로 발생할 수 있습니다. 요청이 너무 복잡하거나 긴 경우, 서버가 일시적으로 과부하 상태인 경우 발생할 수 있습니다. 요청을 더 작은 단위로 나누거나, 잠시 후 다시 시도해보세요.
Rate limit exceeded 에러가 나타납니다
API 사용량 제한을 초과한 경우입니다. 각 API 제공업체는 분당/시간당 요청 수 제한을 두고 있습니다. 잠시 기다렸다가 다시 시도하거나, 계정을 업그레이드하여 더 높은 제한을 받으실 수 있습니다.
연결 오류가 발생했습니다
네트워크 연결을 확인해주세요. 방화벽이나 프록시 설정이 API 호출을 차단하고 있지 않은지 확인하고, 인터넷 연결이 안정적인지 확인해보세요. VPN을 사용 중이라면 일시적으로 비활성화해보세요.
500 Internal Server Error가 발생합니다
서버 측 오류입니다. 이는 일시적인 문제일 가능성이 높으니 잠시 후 다시 시도해보세요. 문제가 지속되면 서비스 상태 페이지를 확인하거나 고객 지원팀에 문의해주세요.
